# woof

woof is Casty genesis OG NFT (ERC-721) contract.

- testnet: base sepolia
- mainnet: degen

## prepare

```bash
## copy and edit .env as needed your environment
copy .env.example .env
npm install
```

## Compile

```bash
npm run build
```

## Deploy

```bash
npm run release
```

## Test

```bash
npm run test
```
